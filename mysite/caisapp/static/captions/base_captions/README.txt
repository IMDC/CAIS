


The caption files under multi-line will incorporate 1-3 English sentences. (some of them will have a single English sentence)

Each caption (and corresponding video clip) content will belong to a genre from a set.
The set will have: sports talkshow, breakfast talkshow, weather forecast, sports broadcasting.

Each caption title will have a form of the following:

v(x)_(genre)-m_(y).vtt, where

- x = clip number
- genre = content genre
- y = caption version, 0 represents the transcript and 100 represents the actual caption generated (paraphrased/edited).
